# LCTA_auto_update
LCTA自动汉化发布脚本&产出

## 使用方法
本项目使用GitHub Actions自动发布。将会自动从[都市零协会汉化组](https://github.com/LocalizeLimbusCompany/LocalizeLimbusCompany)获取最新汉化资源及原文，并自动生成汉化文件，发布至release。  
您可以在[最新release](https://github.com/LocalizeLimbusCompany/LCTA_auto_update/releases/latest)中下载最新汉化文件。

## 许可证

## 二次开发 - 如何为您的语言创建自动翻译？
作为一个自动翻译脚本，LCTA_auto_update理应可以很方便的为任何语言创建自动翻译。但是，由于开发者精力问题，目前仅支持为中文创建自动翻译。如果您有为自己的语言创建自动翻译的需求，可以提交issue或pr。
